The scope of a var variable is a funtional scope. It is global scope when var variable is outside of a function and accessable in the full window. var is functional scope when declared within a function and accesseable only within that function. Var can be redeclare and updated into into scope

The scope of a let variable is block scope. you can update but you can not redeclare in the scope. let can declare without initialization.let cannot be used before declaration. If you declare, it will result in a ReferenceError.

The scope of a const variable is block scope. const cannot be updated and redeclared.value of a variable declared with const remains the same within scope. It cannot be updated or redeclared.
